# 一、产品结构（信息架构 IA）

你的产品本质是一个三合一系统：

1. **字幕平台（核心）**
2. **AI 翻译 + 编辑器（差异化）**
3. **电影信息展示（增强体验）**

👉 推荐结构：

```
AI Movie
├── 首页（搜索）
├── 字幕模块
│   ├── 字幕搜索
│   ├── 字幕详情
│   ├── 字幕编辑器
│
├── 电影模块
│   ├── 电影详情页
│   ├── 分类 / 标签
│
├── 用户模块
│   ├── 收藏
│   ├── 评论
│   ├── 我的字幕
│
├── AI模块
│   ├── AI翻译
│   ├── 实体识别（人名/地名）
│
├── 认证模块
│   ├── 登录 / 注册
```

# 三、核心功能拆解

## 1️⃣ 字幕处理流程

```
上传 SRT
 → 解析
 → 存 DB
 → 展示
 → 编辑
 → AI处理
```

---

## 2️⃣ AI翻译流程

```
SRT
 → chunk (100条)
 → LLM API
 → 合并
 → 写回 DB
```

---

## 3️⃣ 爬虫 / 数据来源

建议不要纯爬：

👉 优先：

- TMDB API
- IMDb 数据

👉 爬虫只做补充：

- 字幕站
- 台词站

---

## 4️⃣ 实体识别（NER）

技术路径：

- Qwen / LLM

输出：

```
{
  text: "Tony is in New York",
  entities: [
    { type: "PERSON", value: "Tony" },
    { type: "LOCATION", value: "New York" }
  ]
}
```

---

## 5️⃣ 时间轴预览

两种方案：

### 简单版

- 滚动高亮

### 高级版

- 视频 + subtitle sync
